# SentinelForge Support

## Free support

For bugs, feature ideas, and questions about using SentinelForge, open a GitHub issue.

Please do not post private code, secrets, `.env` files, passwords, tokens, or client data in public issues.

## Paid Security Baseline Reviews

If you want help using SentinelForge on an app, client project, or AI-built codebase, request a paid Security Baseline Review using the GitHub issue template.

Starter beta pricing:

- Mini review: $49
- Standard review: $149
- Review + GitHub Actions setup: $499

A review can include:

- SentinelForge scan
- Plain-English summary
- Top risks
- Fix checklist
- Release-readiness grade
- Optional GitHub Actions setup

## Best way to ask for help

Open an issue and include:

- What kind of app you are building
- What language/framework it uses
- Whether it is personal, freelance/client, or business software
- Whether you want free usage help or a paid review

Do not include private files or credentials.
