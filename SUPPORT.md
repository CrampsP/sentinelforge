# Support

Use GitHub issues for bugs, feature requests, and questions about running SentinelForge.

For security-sensitive reports, follow `SECURITY.md` and avoid posting secrets or private vulnerability details publicly.

When asking for help, include:

- SentinelForge version
- Operating system
- Command you ran
- Redacted error output
- Whether external scanners such as Semgrep, Bandit, OSV-Scanner, Trivy, or Gitleaks are installed
